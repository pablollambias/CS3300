Problem 2: Each of the following expressions contains improper JSON: calling JSON.parse(...) will result in an error. For each example, describe the error and provide a fixed version. Use the JSON.parse function to demonstrate that it works. Some may contain more than one error.

a.) This contains two errors. First, the input of the JSON.parse function must be a string, so everything in the input must be wrapped by quotes.
The second problem is that the key, i.e. name, is not contained within quotes. The key must be contained within quotes for the function JSON.parse to work.
Corrected Example:
<script>
JSON.parse('{ "name" : "Data-driven web applications" }');
</script>

b.) First, the input of the JSON.parse function must be a string, so everything in the input must be wrapped by quotes. Additionally, one cannot put a comment into the JSON.parse function, as it causes an error. In order to have that comment displayed, one could manipulate the input so that the comment and the name "number" are all in one set of quotes. This entire blurb is now the key for the function. Therefore, we have the following corrected example:

Corrected Example:
<script>
JSON.parse('{ "/* the next variable is a course number */ number": 3300 }');
</script>

Additionally, one can correct it by just removing the comment from the input (perhaps placing it before the actual function call) like so:

<script>
JSON.parse('{ "number": 3300 }');
</script>

c.) First, the input of the JSON.parse function must be a string, so everything in the input must be wrapped by quotes. Second, there is no closing bracket for the list that is being inputted, so a closing bracket must be added at the end of the list. Third, there must be commas in between the three entries of the list. Finally, there is also a semicolon after number in the Python dictionary. That semicolon must be changed to a regular colon.

Corrected Example:
<script>
JSON.parse('[{ "name": "DDWA", "number": 3300 }, { "name": "Java", "number": 2110 }, { "name": "Python", "number": 1110 }]');
</script>
